This project focuses on analyzing banking customer data to gain insights into customer behavior, transaction patterns, account status, and credit categories. The dashboard is built using Python (Pandas) for data processing, SQL for data extraction, and Power BI for interactive data visualization.

Objectives::
Analyze total customers and active accounts
Track total transaction amount and total balance
Compare Credit vs Debit transactions
Identify city-wise customer distribution
Analyze account status (Active, Inactive, Dormant)
Segment customers based on credit score category

Tools & Technologies Used::
Python (Pandas)
SQL
Power BI
Microsoft Excel
GitHub

Future Enhancements::
Add time-based transaction analysis
Integrate live database connection
Implement predictive analysis for customer churn
